Project Introduction

About Project:

This Project is all about redesigning of current COMP 1950 course website.

Current site is not much user friendly as its categorized week based so individual have to walk through or search whole document to find something, e.g., week homework.

Our target audience is:

  • Students (who are taking course)
  • Instructors (who are teaching course)
  • Alumni (who are already passed out but want to visit website for recent topics)
  • Guests (who want to review course content)

Although different audience have different needs so ideally there should be accessibility limitations on each user for which site requires data base functionality. Content will remain same but will be presented in much organized way. Teacher should have access to add, edit or update the content.
